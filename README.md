# HireFellow – AI Enhanced Recruitment Platform

HireFellow is an **AI-powered recruitment platform** that automates resume screening, skill assessment, and behavioral interviews using **Machine Learning and NLP**.

This project aims to reduce hiring time, improve accuracy, and support fair, data-driven recruitment decisions.

---

## Features

- AI-based resume screening using keyword extraction  
- Automated candidate shortlisting  
- Online skill assessment with auto evaluation  
- AI-powered behavioral interview (facial expression analysis)  
- Role-based access (Admin, HR, Candidate)  
- Recruiter analytics and reports
